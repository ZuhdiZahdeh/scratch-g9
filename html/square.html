A) المربع — الطريقة الأسهل

أقل عدد ممكن من اللبنات: حلقة تكرار 4 + حركة ودوران.

كود سكراتش لرسم مربع

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(-225) y:(160)
pen down
repeat (4)
        move (100) steps
        turn clockwise (90) degrees
end
pen up

B) المربع — باستخدام النقاط الثابتة (X/Y)

بدلاً من الحركة والدوران، يمكن الاعتماد على تغيير إحداثيات (X/Y) لرسم الشكل.

كود سكراتش لرسم مربع باستخدام إحداثيات X/Y

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(0) y:(0)
pen down
        go to x:(100) y:(0)
        go to x:(100) y:(100)
        go to x:(0) y:(100)
        go to x:(0) y:(0)
pen up

C) المربع — استخدام متغيرات لضبط طول الضلع

يمكن استخدام متغير للتحكم بمرونة في طول الضلع.

D) ترقية: لبنة مخصّصة — ارسم مربع(الضلع)

يمكن تبسيط الكود وإنشاء لبنة خاصة بنا (لبنة مخصصة) تجمع الأوامر المتكررة.

تعريف اللبنة

define ارسم مربع (الضلع)
repeat (4)
        move (الضلع) steps
        turn clockwise (90) degrees
end

الاستخدام (رسم مربع واحد)

when green flag clicked
erase all
pen down
ارسم مربع (150)
pen up

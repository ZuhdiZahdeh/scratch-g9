A) المعين — باستخدام النقاط الثابتة (X/Y)

استخدم أربع نقاط أركان لرسم معين:

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(-150) y:(0)
pen down
glide (1) secs to x:(0) y:(200)
glide (1) secs to x:(150) y:(0)
glide (1) secs to x:(0) y:(-200)
glide (1) secs to x:(-150) y:(0)
pen up

B) المعين — باستخدام متغيّرين (طول-الضلع، الزاوية-الحادة)

استخدم متغيرات لطول الضلع والزاوية الحادة.

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(-225) y:(160)
set [طول_الضلع v] to (150)
set [الزاوية_الحادة v] to (60)
set [الزاوية_المنفرجة v] to (180 - (الزاوية_الحادة))
pen down
repeat (2)
        move (طول_الضلع) steps
        turn clockwise (الزاوية_الحادة) degrees
        move (طول_الضلع) steps
        turn clockwise (الزاوية_المنفرجة) degrees
end
pen up

C) ترقية: لبنة مخصّصة — ارسم المعين(طول-الضلع، الزاوية-الحادة)

تعريف اللبنة

define ارسم معين (الضلع) (الزاوية_الحادة)
set [الزاوية_المنفرجة v] to (180 - (الزاوية_الحادة))
repeat (2)
        move (الضلع) steps
        turn clockwise (الزاوية_الحادة) degrees
        move (الضلع) steps
        turn clockwise (الزاوية_المنفرجة) degrees
end

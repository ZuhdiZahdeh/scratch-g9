الدرس ٣: زخارف سداسية والتعميم بالمتغيّرات

نرسم ٦ أشكال سداسية بزاوية دوران خارجية 360/6 = 60°، ثم نعمّم الفكرة إلى زخارف عامة بالمتغيّرات (عدد الأضلاع، الطول، عدد التكرارات، تغيّر اللون).

الرئيسية الدرس السابق الدرس التالي

الأهداف التعليمية

  • استخدام لبنات القلم Pen مع التكرار لرسم مضلعات وزخارف.
  • فهم العلاقة زاوية الدوران الخارجية = 360 ÷ عدد الأضلاع.
  • بناء تكرارات متداخلة (حلقة داخل حلقة) والتحكّم باللون والسرعة.
  • تعميم الحل باستخدام متغيّرات ولبنة مخصّصة (My Blocks).

النشاط الرئيس: ٦ سداسيات تدور 60°

الفكرة: نرسم سداسيًا واحدًا، ثم نكرّر رسم السداسي ٦ مرات، ومع كل مرة نستدير 60° لنحصل على شكل الزخرفة.

لماذا 60°؟

الدائرة 360°. لتوزيع ٦ أشكال بالتساوي حول المركز ⇒ 360/6 = 60°. وهي أيضًا الزاوية الخارجية للسداسي المنتظم.

الكود المقترح (حل النشاط)

when green flag clicked erase all go to x: (0) y: (0) set pen size to (2) pen down repeat (6) { // السداسي 6 مرات حول المركز   change pen color by (30)   repeat (6) { // رسم سداسي واحد     move (50) steps     turn right ((360) / (6)) degrees   }   turn right ((360) / (6)) degrees // توزيع 6 سداسيات حول المركز   wait (0.3) seconds // إبطاء لإظهار كل سداسي } pen up

لإبطاء الرسم، اجعل انتظر داخل الحلقة الخارجية (بعد كل سداسي). ولإظهار كل ضلع ببطء شديد ضع الانتظار داخل الحلقة الداخلية.

أسئلة على النشاط

  1. أين تضع لبنة انتظر لتبطئ الرسم؟ ولماذا هذا الموضع؟
  2. ما التكرار الذي يرسم شكلاً سداسيًا واحدًا؟ وما التكرار الذي يرسم ستة سداسيات؟
  3. ما زاوية الاستدارة داخل السداسي؟ ولماذا لا نستخدم 120° في الاستدارة الداخلية؟
  4. اكتب خوارزمية سير البرنامج خطوة بخطوة.

التعميم: زخارف عامة بالمتغيّرات + My Blocks

نُعرّف المتغيّرات: n (الأضلاع)، len (الطول)، k (عدد النسخ حول المركز)، cStep (تغيّر اللون)، وننشئ لبنة مخصّصة لرسم مضلع عام.

١) لبنة رسم مضلع عام

define DrawPolygon (sides n) (length len) repeat (n)   move (len) steps   turn right ((360) / (n)) degrees end

٢) زخرفة عامّة حول المركز

when green flag clicked set [n v] to (6) set [len v] to (50) set [k v] to (9) set [cStep v] to (20) erase all pen down repeat (k)   change pen color by (cStep)   DrawPolygon (n) (len)   turn right ((360) / (k)) degrees   wait (0.2) seconds end pen up

بدّل قيم n, len, k, cStep لتوليد زخارف مختلفة.

تمارين إثرائية

  • بدّل k إلى 8 ثم 9. ما زاوية الدوران المناسبة كل مرة؟ (الإجابة: 360/k).
  • استبدل السداسي بمربّع: n=4 وجرّب k=4 ثم k=8.
  • ارفع cStep إلى 30–40 أو اجعله سالبًا لتدرّج عكسي.

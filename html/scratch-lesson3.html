الدرس ٣: زخارف سداسية والتعميم بالمتغيّرات

نرسم ٦ أشكال سداسية متتالية بزاوية دوران 360/6 = 60°، ثم نعمّم الفكرة إلى زخارف عامة بالمتغيّرات (عدد الأضلاع، الطول، عدد التكرارات، تغيّر اللون).

الرئيسية الدرس السابق الدرس التالي

الأهداف التعليمية

  • استخدام لبنات القلم Pen مع التكرار لرسم مضلعات وزخارف.
  • فهم العلاقة زاوية الدوران الخارجية = 360 ÷ عدد الأضلاع.
  • بناء تكرارات متداخلة (حلقة داخل حلقة) والتحكّم باللون والسرعة.
  • تعميم الحل باستخدام متغيّرات ولبنة مخصّصة (My Blocks).

النشاط الرئيس: ٦ سداسيات تدور 60°

الفكرة: نرسم سداسيًا واحدًا (٦ أضلاع، كل ضلع حركة + استدارة خارجية 60°)، ثم نكرّر رسم السداسي ٦ مرات، ومع كل مرة نستدير 60° لنحصل على شكل الزخرفة.

لماذا 60°؟

الدوران الكامل دائرة = 360°. إذا أردنا توزيع ٦ أشكال بالتساوي حول المركز، نقسم الدوران على ٦ ⇒ 360/6 = 60°. كذلك الزاوية الخارجية لمضلع سداسي منتظم هي 360/6 = 60°، لذا نستعملها في الاستدارة داخل حلقة رسم السداسي.

الكود المقترح (حل النشاط)

when green flag clicked erase all go to x: (0) y: (0) set pen size to (2) pen down repeat (6) { // السداسي 6 مرات حول المركز   change pen color by (30)   repeat (6) { // رسم سداسي واحد     move (50) steps     turn right ((360) / (6)) degrees   }   turn right ((360) / (6)) degrees // توزيع 6 سداسيات حول المركز   wait (0.3) seconds // إبطاء لإظهار كل سداسي } pen up

ملاحظة “إبطاء الرسم”: ضع انتظر 0.3 ثانية داخل الحلقة الخارجية (بعد الانتهاء من السداسي) إن أردت التوقّف بين كل سداسي وآخر. ضعها داخل الحلقة الداخلية إذا أردت إظهار رسم كل ضلع ببطء شديد.

أسئلة على النشاط

  1. لتصبح عملية الرسم بطيئة: أين تضيف لبنة انتظار؟ ولماذا اخترت هذا الموضع؟
  2. ما التكرارات المتداخلة في البرنامج؟ ما التكرار الذي يرسم شكلاً سداسيًا واحدًا؟ وما التكرار الذي يرسم ستة سداسيات؟
  3. تتكوّن خوارزمية رسم السداسي من: تكرار خطوتين (تحرّك ثم استدر). ما زاوية الدوران؟ ولماذا لا نستخدم 120° في الاستدارة الداخلية؟
  4. اكتب خوارزمية (خطوات) سير البرنامج من البداية حتى اكتمال الشكل.

التعميم: تصميم زخارف عامة بالمتغيّرات + My Blocks

نعرف متغيّرات للتحكّم بالشكل: الأضلاع n، الطول len، عدد التكرارات k (عدد المضلعات حول المركز)، تغيّر اللون cStep. ونبني لبنة مخصّصة ترسم مضلعًا عامًّا.

١) إنشاء لبنة مخصّصة

define DrawPolygon (sides n) (length len) repeat (n)   move (len) steps   turn right ((360) / (n)) degrees end

٢) زخرفة عامّة حول المركز

when green flag clicked set [n v] to (6) set [len v] to (50) set [k v] to (9) set [cStep v] to (20) erase all pen down repeat (k)   change pen color by (cStep)   DrawPolygon (n) (len)   turn right ((360) / (k)) degrees   wait (0.2) seconds end pen up

بدّل قيم n, len, k, cStep لتوليد زخارف مختلفة (نجميّة/وروديّة/حلقات…).

تمارين إثرائية

أ) غيّر توزيع الأشكال

اجعل k = 8 ثم k = 9. ما زاوية الدوران المناسبة في كل مرة؟

الإجابة: 360/k.

ب) مربّعات تدور

استبدل السداسي بمربّع (n = 4). ارسم ٤ مربعات حول المركز (k = 4)، ثم ٨ مربعات.

ج) دوّارة ألوان

حرّك cStep إلى قيم كبيرة (مثلاً 30–40) أو سالبة لصنع تدرّج عكسي.

إجابات نموذجية مختصرة

  • موضع الانتظار: بعد الانتهاء من كل مضلع في الحلقة الخارجية لإظهار فرق بصري بين الأشكال.
  • التكرارات: الداخلية ترسم السداسي (٦ مرات). الخارجية تكرّر السداسي ٦ مرات مع دوران 60°.
  • لماذا ليست 120°؟ 120° زاوية داخلية للمثلث عند رسمه؛ أمّا زاوية الاستدارة الخارجية للمضلع المنتظم فهي 360/n (سداسيًا: 60°).
  • خوارزمية السير: مسح ← موقع البداية ← إنزال القلم ← (كرّر k: غيّر اللون ← (كرّر n: تحرّك/استدر 360/n) ← استدر 360/k ← انتظر) ← ارفع القلم.

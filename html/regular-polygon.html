رسم الشكل المنتظم (n-gon)

← العودة إلى قائمة الأشكال

A) برنامج عام — أي مضلع منتظم (n-gon)

تعريف اللبنة

define ارسم مضلع منتظم (n) (L)
set [زاوية v] to (360 / (n))
repeat (n)
        move (L) steps
        turn clockwise (زاوية) degrees
end

الاستخدام بسؤالين للمستخدم

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(0) y:(160)
ask [كم عدد الأضلاع؟] and wait
set [n v] to (answer)
ask [ما طول الضلع؟] and wait
set [L v] to (answer)
pen down
ارسم مضلع منتظم (n) (L)
pen up

مثال: خماسي → زاوية = 360/5 = 72°، مثمّن → 45°.

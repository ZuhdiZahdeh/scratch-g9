من المربع البسيط → المعين → أي مضلع منتظم (n-gon)

نتدرّج من أبسط برنامج لرسم مربع، ثم نعيد الصياغة باستخدام المتغيّرات واللبنات المخصّصة، ونطوّر الفكرة لرسم معيّن ثم أي مضلع منتظم.

A) المربع — الطريقة الأسهل

أقل عدد ممكن من اللبنات: حلقة تكرار 4 + حركة ودوران.

كود سكراتش لرسم مربع

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(-225) y:(160)
pen down
repeat (4)
        move (320) steps
        turn clockwise (90) degrees
end
pen up

النتيجة النهائية لرسم المربع

يمكن تغيير 320 حسب المساحة المتاحة على المسرح.

B) المربع — باستخدام النقاط الثابتة (X/Y)

استخدم أربع نقاط أركان: (-230,160) → (230,160) → (230,-160) → (-230,-160) → العودة.

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(-230) y:(160)
pen down
glide (1) secs to x:(230) y:(160)
glide (1) secs to x:(230) y:(-160)
glide (1) secs to x:(-230) y:(-160)
glide (1) secs to x:(-230) y:(160)
pen up

C) المربع — باستخدام متغيّر (طول_الضلع)

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(-225) y:(160)
set [طول_الضلع v] to (150)
pen down
repeat (4)
        move (طول_الضلع) steps
        turn clockwise (90) degrees
end
pen up

D) ترقية: لبنة مخصّصة — ارسم مربع(الضلع)

تعريف اللبنة

define ارسم مربع (الضلع)
repeat (4)
        move (الضلع) steps
        turn clockwise (90) degrees
end

الاستخدام

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(-225) y:(160)
pen down
ارسم مربع (150)
pen up

E) تصميم المستطيل

يمكنك رسم مستطيل باستخدام ضلعين مختلفين في الطول.

كود سكراتش لرسم مستطيل

لإنشاء مستطيل باستخدام الحركة والدوران، يجب على الكائن أن يرسم ضلعين متجاورين بطولين مختلفين، ثم يكرر هذه العملية.

النتيجة النهائية لرسم المستطيل

F) المعين — لبنة عامة: ارسم معين(الضلع، الزاوية_الحادّة)

يمكنك رسم المعين باستخدام أضلاع متساوية في الطول ولكن بزوايا مختلفة عن 90 درجة.

define ارسم معين (الضلع) (الزاوية_الحادة)
set [الزاوية_المنفرجة v] to (180 - (الزاوية_الحادة))
repeat (2)
        move (الضلع) steps
        turn clockwise (الزاوية_الحادة) degrees
        move (الضلع) steps
        turn clockwise (الزاوية_المنفرجة) degrees
end
when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(0) y:(160)
pen down
ارسم معين (150) (60)
pen up

G) برنامج عام — أي مضلع منتظم (n-gon)

تعريف اللبنة

define ارسم مضلع منتظم (n) (L)
set [زاوية v] to (360 / (n))
repeat (n)
        move (L) steps
        turn clockwise (زاوية) degrees
end

الاستخدام بسؤالين للمستخدم

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(0) y:(160)
ask [كم عدد الأضلاع؟] and wait
set [n v] to (answer)
ask [ما طول الضلع؟] and wait
set [L v] to (answer)
pen down
ارسم مضلع منتظم (n) (L)
pen up

مثال: خماسي → زاوية = 360/5 = 72°، مثمّن → 45°.

واجبات الدرس

ملاحظة هامة لتسليم المشاريع

لتنظيم المشاريع، يجب عليك تسمية ملف المشروع (ملف .sb3) بطريقة موحدة قبل تسليمه.

**صيغة التسمية:** YourName-ProjectName

**مثال:** إذا كان اسمك "Ali" واسم المشروع هو "Hello World"، يكون اسم الملف: **Ali-Hello_World**.

تذكر أن تتبع هذه الصيغة في جميع واجباتك، سواء كنت في المختبر أو في المنزل!

واجب الحصة (للمختبر)

مشروع "القطة الناطقة" **(Talking_Cat)**:

  • افتح مشروعًا جديدًا.
  • اجعل القطة تقول "مرحباً!" ثم تغير لونها، ثم تتحرك 10 خطوات.
  • احفظ المشروع بالصيغة المذكورة أعلاه.

واجب منزلي (للكلاسروم)

مشروع "الرسوم المتحركة" **(Animation_Project)**:

  • أضف كائنين إلى المنصة، وبرمجهما ليتحدثا مع بعضهما البعض.
  • اجعل كل كائن يغير مظهره أثناء الحديث.
  • احفظ المشروع بالصيغة المذكورة وارفعه على الكلاسروم.

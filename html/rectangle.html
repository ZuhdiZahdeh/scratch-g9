رسم المستطيل

← العودة إلى قائمة الأشكال

A) المستطيل — الطريقة الأسهل

أقل عدد ممكن من اللبنات: حلقة تكرار 2 + حركتين ودوران.

كود سكراتش لرسم مستطيل

لإنشاء مستطيل باستخدام الحركة والدوران، يجب على الكائن أن يرسم ضلعين متجاورين بطولين مختلفين، ثم يكرر هذه العملية.

النتيجة النهائية لرسم المستطيل

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(-225) y:(160)
pen down
repeat (2)
        move (300) steps
        turn clockwise (90) degrees
        move (150) steps
        turn clockwise (90) degrees
end
pen up

B) المستطيل — باستخدام النقاط الثابتة (X/Y)

استخدم أربع نقاط أركان: (-230,160) → (230,160) → (230,-160) → (-230,-160) → العودة.

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(-230) y:(160)
pen down
glide (1) secs to x:(230) y:(160)
glide (1) secs to x:(230) y:(-160)
glide (1) secs to x:(-230) y:(-160)
glide (1) secs to x:(-230) y:(160)
pen up

C) المستطيل — باستخدام متغيّرين (طول-الضلع، عرض-الضلع)

when green flag clicked
erase all
pen up
set pen size to (5)
go to x:(-225) y:(160)
set [طول_الضلع v] to (300)
set [عرض_الضلع v] to (150)
pen down
repeat (2)
        move (طول_الضلع) steps
        turn clockwise (90) degrees
        move (عرض_الضلع) steps
        turn clockwise (90) degrees
end
pen up

D) ترقية: لبنة مخصّصة — ارسم المستطيل(الطول، العرض)

تعريف اللبنة

define ارسم المستطيل (طول) (عرض)
repeat (2)
        move (طول) steps
        turn clockwise (90) degrees
        move (عرض) steps
        turn clockwise (90) degrees
end

when green flag clicked
erase all
pen up

go to x:(-225) y:(160)
pen down
ارسم المستطيل (300) (150)
